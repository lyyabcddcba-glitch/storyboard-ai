# AI招生增长系统 — 上线指南

## 路线图：从本地到微信小程序

```
你现在的代码
    │
    ├─→ ① 部署网页到公网（Vercel）      ← 10分钟
    │
    ├─→ ② 注册微信小程序账号            ← 5分钟（已有则跳过）
    │
    ├─→ ③A 方案A：web-view 套壳        ← 最快，1小时
    │    （打开微信开发者工具 → 导入 miniapp/ → 改 URL → 上传）
    │
    └─→ ③B 方案B：原生小程序页面        ← 备选，无需服务器
         （打开微信开发者工具 → 导入 miniapp/ → 直接预览）
```

---

## 第一步：部署网页到 Vercel（获得公网 URL）

### 1.1 安装并登录 Vercel

```bash
# 在项目目录下运行
cd d:\first-cc\first
npx vercel login
```

浏览器会自动打开，用 GitHub / GitLab / Email 注册或登录。

### 1.2 部署

```bash
npx vercel
```

按提示操作（全部选默认即可）：
- Set up and deploy? → **Y**
- Which scope? → **选择你的账号**
- Link to existing project? → **N**
- Project name? → **回车（用默认）**
- Root directory? → **回车（用 ./）**
- Build settings override? → **N**

等待 30 秒，部署完成。CLI 会输出一个 URL，类似：
```
https://ai-enrollment-xxxxx.vercel.app
```

### 1.3 （可选）自定义域名

在 Vercel 后台 → Settings → Domains → 添加你的域名。

---

## 第二步：注册微信小程序

1. 打开 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击右上角 "立即注册" → 选择 "小程序"
3. 填写邮箱、密码，激活账号
4. 选择主体类型：
   - **个人**：免费，但 web-view 业务域名受限（可以用方案B）
   - **企业**：需营业执照，功能完整
5. 注册完成后，进入后台 → 开发管理 → 开发设置 → 复制 **AppID**

---

## 第三步：配置小程序项目

### 3.1 下载微信开发者工具

https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

安装后打开。

### 3.2 导入项目

1. 微信开发者工具 → 点击 "+" 或 "导入项目"
2. 项目目录：选择 `d:\first-cc\miniapp\`
3. AppID：粘贴第二步复制的 AppID
4. 项目名称：AI招生增长系统
5. 点击 "导入"

---

## 第四步：选择你的方案

### 方案A：web-view 套壳（推荐，前提：已部署公网URL）

**适用条件**：网页已部署到 Vercel（第一步完成）

1. 打开 `miniapp/pages/index/index.js`
2. 修改第 3 行的 URL：
   ```js
   webUrl: 'https://你的Vercel地址.vercel.app'
   ```
3. 在微信小程序后台 → 开发管理 → 开发设置 → 业务域名
   - 添加你的 Vercel 域名
   - 下载校验文件，放到 `d:\first-cc\first\public\` 目录
   - 重新部署 Vercel（`npx vercel --prod`）
4. 在微信开发者工具中，点击 "编译" → 预览

**优点**：网页版所有功能直接可用，改一处两端同步
**缺点**：需要配置业务域名（企业主体）

---

### 方案B：原生小程序页面（推荐，个人主体也能用）

**适用条件**：无需服务器，无需域名，无需企业主体

1. 打开 `miniapp/app.json`
2. 确保 `pages/native/native` 在 pages 数组的第一位（已默认设置）
3. 在微信开发者工具中，点击 "编译"
4. 直接就能看到效果！

**优点**：
- 零服务器成本
- 个人主体直接可用
- 打开速度极快（原生渲染）
- 无需域名配置

**缺点**：
- 只有销售话术模块（最核心的）
- 数据是静态的（已预置4个痛点12套方案）
- 更新内容需要重新上传小程序

---

## 第五步：生成体验版二维码

1. 在微信开发者工具中，点击右上角 "上传"
2. 版本号填 `0.1.0`，项目备注填 "第一版体验"
3. 上传成功后，打开 [微信小程序后台](https://mp.weixin.qq.com/)
4. 左侧菜单 → 管理 → 版本管理
5. 在 "开发版本" 中找到刚上传的版本，点击右侧 "选为体验版"
6. 生成 **体验版二维码**

---

## 第六步：发给用户测试

### 6.1 添加体验者

在微信小程序后台 → 管理 → 成员管理 → 添加 10 个用户的微信号为 "体验成员"。

### 6.2 发二维码

把体验版二维码发到微信群或私发。用户扫码 → 打开小程序 → 直接看效果。

### 6.3 收集反馈

观察以下几点：
- 他们打开后第一反应是什么？
- 他们会主动点哪个按钮？
- 他们会不会复制话术去用？
- 他们有没有问 "这个怎么收费？"

如果 10 个人里有 3 个人问 "怎么收费" → 说明愿意付费。

## 绿野仙踪测试法（0成本验证）

如果你连小程序都不想注册，最快验证方式：

1. 建一个微信群 "AI招生智库内测群"
2. 拉 5 个驾校销售/教培校长
3. 群公告写：*"遇到家长刁钻问题，直接发群里，AI 秒回你方案"*
4. 他们发问题 → 你打开 `http://localhost:3000` → 输入问题 → 复制结果 → 发回群里
5. 假装是 AI 自动回复

跑 3 天。如果他们天天在群里 @你帮出方案 → **需求刚需，立即投入**。
